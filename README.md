This is a demo running dnode and scuttlebutt on the same websocket stream with mux-demux.

Code is writting in CoffeeScript.

# Install #

```
sudo npm install grunt-cli browserify -g
npm install
```

then complie:

```
grunt
```

# Run #

Start server:

```
node lib/server.js
```

Start browser: `http://localhost:9999`
